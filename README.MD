## Переменные окружения

| Переменная окружения | Описание                     |
|----------------------|------------------------------|
| `DB_URL`             | URL-адрес базы данных        |
| `DB_USERNAME`        | Имя пользователя базы данных |
| `DB_PASSWORD`        | Пароль от базы данных        |
| `JWT_SECRET`         | Секретный ключ JWT           |
